How to Use The procedure on how to use our service.

Upload audio file

For uploading an audio file go to the home page and then select the choose file option displayed below to upload the file required file from your system and click on the upload button displayed below.

Apply filters

A filter is a device or process that removes some unwanted components or features from a signal. Filtering is a class of signal processing, the defining feature of filters being the complete or partial suppression of some aspect of the signal. To apply filters for the uploaded audio file, click on the filters option displayed on the top left corner of the page. There you can choose the filter you want to apply and click on the apply option displayed below.

Play audio file

After applying filter to your audio file, it redirects you in to a new web page where you can play and hear the audio file.
