What are filters?

Different audio filers are used in everyday day life which depend on the sound frequency. There are many types of filters for each type of filter the frequency of the audio files changes.

Low Pass filter

This particular filter when applied the audio file the user can hear the low frequency content(below the cutoff frequency) of the audio file, the high frequency waves are removed and user can only listen to the low frequency.

High Pass Filter

T This particular filter when applied to audio file the user can hear the high frequency content(above the cutoff frequency) of the audio file,the low frequecy waves are removed and user can only listen to the high frequency.

Echo Filter

The user is able to listen to the repated frequencies of content in the audio file after certain amount of delay. It adds and echo effect to the audio file and you can hear an echo produced by the filter.
